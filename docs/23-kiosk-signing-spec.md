# 23 — 手持簽署確認系統（kiosk signing）規格

> 2026-07-05 與店主逐點確認定案。客人面向的手持裝置（手機/平板）簽署系統：
> 收購切結書、購物金扣抵確認、交易紀錄簽收。**與 POS 分離的獨立頁面、共用後端**。

## 1. 核心原則：簽名綁定內容快照

簽名永遠綁定「當下畫面上的品項、金額、個資、條款版本、撥款選擇」的 **JSONB 快照**。
任何變更（改價/換撥款/換人）＝舊任務作廢、重推重簽。改價、反悔、事後爭議共用同一機制。

## 2. 收購簽署流

```
0. 建檔        櫃檯 contacts 頁（含新增住址欄）
1. 估價        櫃檯收購頁照現行填品項/估價（未入庫、隨時改）→「送出至手持裝置」建任務（快照）
2. 手持選人    待機畫面自動顯示最新待簽任務；店員可搜姓名/手機切換任務
3. 客人確認    單頁：① 個資（姓名/手機/證號遮罩/住址；「資料有誤」退回櫃檯改）
               ② 切結書＋條款全文（版本化，§5）③ 品項逐項售價
4. 選撥款      現金 / 購物金 二選一（顯示現有餘額＋入帳後餘額預覽含溢價）；簽名前隨時可改
5. 簽名        點簽名區→全螢幕放大→清除重簽→送出；櫃檯即時顯示「已簽署 ✓」
               → 店員按「完成收購」→ 後端驗「已簽任務快照 == 送出內容」才建收購（原子）
6. 列印留存    EPSON 印收購憑證聯（品項/金額/撥款/剩餘購物金/簽名影像）；標籤機印商品標籤
               系統留存：快照＋簽名圖＋條款版本＋時間戳，綁 acquisition_id
```

**反悔機制**：簽名前隨時改；簽後、店員按「完成收購」前 → 櫃檯作廢任務重來（主要反悔窗口，
故「完成收購」由店員按、不自動）；收購完成後 → 走既有 F6.5 作廢收購（對稱反轉、留稽核）重做。

## 3. 購物金使用流

POS 選購物金收款 → 自動推「扣抵確認」任務 → 手持顯示「本次折抵 $X、剩餘 $Y」＋可展開
歷史明細 → 簽名 → 櫃檯放行結帳。客人要印明細且用了購物金 → 明細聯加印剩餘購物金＋簽名影像。
交易紀錄簽收：自 /sales 或會員頁推「交易紀錄確認」任務 → 客人核對 → 簽名留存。

## 4. 架構

- 後端新 `signing` 模組：
  - `signature_tasks`：store_id、kind（ACQUISITION_AFFIDAVIT / STORE_CREDIT_USE /
    TRANSACTION_ACK）、status（PENDING→SIGNED / CANCELLED / EXPIRED）、content JSONB 快照、
    agreement_version、signature_image（PNG bytea）、signed_at、ref（acquisition_id/sale_id/
    contact_id）、created_by
  - `agreement_versions`：切結書/條款版本化全文（現行版＋歷史；改版不影響舊簽名指向）
- `UserRole` 加 `KIOSK`；deps 層限定其可及端點（簽署任務 CRUD、會員基本查詢、購物金唯讀）
- 前端 `/kiosk` 獨立 route group（無店務導覽殼），2 秒輪詢任務（單店 LAN，不上 WebSocket）
- 收購 `POST /acquisitions` 新增 `signature_task_id`（強制，D2）；撥款方式自任務快照回填（D7）
- hardware-agent：列印明細延伸簽名影像與剩餘購物金欄（D6）；簽名 PNG → ESC/POS raster
- 簽名影像入 DB、隨 C6 邏輯備份

### 二手商品讓售切結書 暨 個人資料告知同意書（v1）

出賣人（以下稱「本人」）茲將本文件所列物品讓售／寄售予【店家名稱】（以下稱「本店」），
並聲明及同意下列各條款：

**一、物品來源保證（非贓物切結）**
本人切結保證所讓售／寄售之物品均為本人合法取得、擁有完整處分權，絕非贓物、
侵占、詐欺所得或其他來路不明之物。如日後經司法或檢警機關認定為贓物或涉及不法，
本人願無條件配合調查，自負全部民事、刑事及行政責任，並賠償本店因此所受之一切損失
（含但不限於買賣價金、營業損失、訴訟及律師費用）。

**二、交易確認**
本人已逐項確認本文件所列物品之品名、數量與讓售金額（或寄售抽成比例），並同意以
本人於手持裝置上所選擇之方式（現金或購物金）受領價金。金額經本人簽名確認後，
除本店同意外，不得事後異議。

**三、售出概不退還**
本人瞭解並同意：物品一經讓售並完成點交，所有權即移轉本店，概不退還、不得要求買回。
寄售物品依寄售約定辦理。

**四、瑕疵告知**
本人保證已就物品之瑕疵、損壞、維修紀錄等如實告知本店；如有隱瞞致本店受損，
本人願負賠償責任。

**五、個人資料告知與同意（個人資料保護法第 8 條）**
本店蒐集本人之姓名、聯絡電話、身分證統一編號、住址等個人資料，僅作為
**交易紀錄留存、會員管理、依法令要求之用途**使用，保存於本店營運所需期間。
除經司法、檢調或其他有權機關依法調閱外，絕不提供任何第三人，亦不作其他目的之利用。
本人得依個資法請求查詢、閱覽、補正或刪除本人資料（依法令應留存之交易紀錄除外）。

**六、其他**
本人同意本店以電子方式（手持裝置簽名）簽署本文件，其效力與書面簽名相同；
本文件如有未盡事宜，依中華民國法律及誠信原則辦理。
